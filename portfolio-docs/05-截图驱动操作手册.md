# 截图驱动操作手册

当前截图资产还没有完全重拍，因此这份文档只保留“现行流程说明”，不再把历史截图文件名当成产品事实来源。

## 当前应展示的步骤

1. 首页输入主题，配置文体、风格、受众、字数与 TN 开关。
2. 上传补充资料。
3. 进入研究阶段。
4. 选择讨论方向。
5. 审阅大纲。
6. 启动分段写作。
7. 查看终稿与 `writing_insights.md`、`evidence_cards.md`、`critique.md`。
8. 用 Copilot 或选区改写继续微调。
9. 导出正文和可选 TN。

## 截图更新原则

- 新截图必须围绕商业文章流程拍摄。
- 截图标题与注释统一使用当前术语：
  - 讨论方向
  - 文章大纲
  - 分段写作
  - 终稿审查
  - TN / 讨论指南
- 不再沿用旧的教学型文档术语。
