# 运行部署与验证手册

## 本地运行

```bash
npm install
npm run dev
```

## 构建验证

```bash
npm run build
```

## 基础验收项

1. 首页能输入主题并配置任务参数。
2. 能正常调用研究流程。
3. 能生成 5 个讨论方向。
4. 能生成并审阅大纲。
5. 能完成分段写作并输出终稿。
6. 可选 TN 能按开关决定是否出现。
7. 结果页能继续做改写和终稿清洗。
8. 正文 PDF 能正常导出。

## 回归检查

- active 代码中不再使用旧的 case 流程语义。
- prompt 规则来自项目内 Markdown 资产。
- 长文按 chunk 机制运行，不再强依赖单次直出。
