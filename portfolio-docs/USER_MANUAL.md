# Writing Workspace 作品集手册

这份手册用于作品集演示，和根目录的 `USER_MANUAL.md` 保持同一条流程：

1. 输入主题并配置任务。
2. 让系统完成搜索研究和资料整理。
3. 选择讨论方向。
4. 审阅大纲。
5. 进入分段写作。
6. 审查终稿。
7. 按需导出正文与 TN。

如果需要更完整的使用说明，请直接查看根目录的 [`USER_MANUAL.md`](../USER_MANUAL.md)。
