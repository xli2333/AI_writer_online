# Writing Workspace Portfolio Docs

这里保存的是面向作品集和交付说明的辅助文档，内容已经同步到当前的商业文章写作工作流。

当前产品定位：

- 不是案例生成器
- 不是教学目标驱动流程
- 是一套面向商业文章的研究、规划、成文与审稿工作台

当前主流程：

`选题 -> 搜索研究 -> 整理资料 -> 选择讨论方向 -> 审阅大纲 -> 分段成文 -> 审查 -> 导出`

TN / 讨论指南保留，但由任务配置控制，默认不是强制产物。
