# 华尔街见闻风格反模式
- 不要写成券商研报摘抄、市场传言转发或群聊观点汇总，必须给出自己的信息组织和判断主线。
- 不要只报涨跌和数字，要解释价格背后的政策传导、资金路径、预期变化和交易拥挤度。
- 不要把宏观稿写成空泛大词堆砌，必须落到利率、汇率、信用、商品、风险偏好等具体传导链条。
- 不要把公司稿写成财报流水账，必须回答增长从哪里来、利润由什么支撑、估值为何重定价。
- 不要把策略稿写成口号式看多看空，至少给出情景、触发条件、关键变量和失效点。
- 不要把会员导流、课程宣传、活动通知和周度日程文案当成正式报道口吻复现。
- 不要用“市场认为、机构普遍预计、资金正在博弈”这类模糊主体替代真实来源和具体动作。
