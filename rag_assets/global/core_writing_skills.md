# 核心写作总纲

## 写作目标

**目标受众:** 商业决策者、行业研究者、管理者、对商业趋势和企业实践感兴趣的读者。
**核心目的:**
1.  **洞察与分析:** 深入剖析商业现象、企业战略、管理模式、技术趋势，提供前瞻性、专业性的解读。
2.  **知识传递:** 系统性地梳理和解释复杂的商业理论、概念和模型。
3.  **经验借鉴:** 通过案例分析，提炼成功的商业实践和可复制的经验。
4.  **启迪思考:** 激发读者对商业世界更深层次的理解和对未来趋势的思考。
5.  **价值输出:** 最终为读者提供可行动的见解、决策支持或战略启发。

## 标题原则

1.  **信息明确:** 标题应直接反映文章核心内容，让读者快速了解文章主题。
2.  **吸引力强:** 结合数字、疑问、比喻、冲突、热点词汇等，激发读者阅读兴趣。
3.  **聚焦价值:** 突出文章能为读者带来的核心价值、洞察或答案。
4.  **简洁有力:** 避免冗长，力求精炼，易于记忆和传播。
5.  **准确性:** 标题与内容高度契合，不夸大、不误导。
6.  **风格统一:** 在不同类型文章中，标题风格可调整，但需符合整体品牌调性。

*   **示例:**
    *   **深度分析类:** "XX行业XX趋势深度拆解：XX巨头的XX战略解析"
    *   **案例分析类:** "XX企业XX年XX之路：XX模式的成功与启示"
    *   **人物专访类:** "XX CEO张三：XX产品背后的XX管理哲学"
    *   **趋势预测类:** "AI Agent时代：XX行业XX年后的XX面貌"
    *   **理论解读类:** "颠覆管理学：XX新理论如何重塑XX企业"

## 开头原则

1.  **快速切入主题:** 开篇即点明文章核心观点、研究对象或要解决的问题。
2.  **制造悬念/兴趣:**
    *   **设问:** 以一个引人深思的问题开篇，激发读者好奇心。
    *   **数据/事实:** 抛出震撼性的数据、现象或研究发现。
    *   **比喻/类比:** 用生动形象的比喻或与熟悉事物进行类比，拉近与读者的距离。
    *   **故事/事件:** 以一个引人入胜的案例、故事或历史事件开篇。
    *   **引用:** 引用权威人士、经典著作或名言，建立权威性。
3.  **明确文章价值:** 简要说明文章将提供什么信息、分析什么内容、带来什么启示。
4.  **铺垫背景:** 在必要时，简要介绍相关背景信息，帮助读者理解后续内容。
5.  **结构预告:** （可选）简要勾勒文章的结构框架，让读者对内容有整体认知。

*   **示例:**
    *   **问题开篇:** "当竞争对手敲钟上市，为什么你可以‘开香槟’？这背后隐藏着商业世界一个被忽视的秘密。"
    *   **数据开篇:** "2023年，全球AI投入突破XXX亿美元，但真正理解AI的‘管理密码’的企业屈指可数。"
    *   **故事开篇:** "9年前，《疯狂动物城》横空出世，9年后，其续集再次在中国市场掀起观影热潮，并带动了周边产品的‘无用溢价’。这背后，藏着迪士尼IP营销的深层逻辑。"
    *   **理论开篇:** "管理学的黄昏与智能的黎明——陈天桥这篇万字长文，正在颠覆我们对企业和管理的认知。"

## 结构原则

1.  **总-分-总结构:**
    *   **总:** 开篇确立核心论点、提出问题或概括主题。
    *   **分:** 分层级、分维度地展开论证，每个部分围绕总论点展开，并提供详实的论据、案例或数据。
    *   **总:** 结尾总结前文，重申核心论点，升华主旨，或提出未来展望。
2.  **主题式结构:** 将文章内容分解为若干个独立但相互关联的主题模块，每个模块深入探讨一个子议题。
3.  **时间线叙事:** 按照事件发生发展的顺序进行叙述，适用于历史回顾、企业成长史等。
4.  **问答式结构:** 以读者可能关心的问题为导向，逐一解答，适用于专访、FAQ类内容。
5.  **逻辑清晰:** 各部分之间过渡自然，层层递进，论证严谨，避免跳跃和混乱。
6.  **模块化设计:** 采用清晰的小标题、段落划分，使结构一目了然，便于读者阅读和信息检索。
7.  **层级分明:** 运用多级标题（H2, H3, H4...）构建清晰的内容层级。

*   **常用结构模式:**
    *   **总-分-总:** (最常用，适用于论证类、分析类)
    *   **问题-解答 (Q&A):** (适用于访谈、解读)
    *   **时间顺序:** (适用于历史回顾、人物传记)
    *   **主题罗列:** (适用于趋势分析、方法总结)

## 论证原则

1.  **事实为基础:** 所有论点均需有可靠的事实、数据、案例或研究作为支撑。
2.  **逻辑严谨:** 论证过程必须符合逻辑，避免滑坡谬误、以偏概全、因果倒置等逻辑错误。
3.  **多角度分析:** 从不同维度（宏观、微观、技术、市场、管理、文化等）对问题进行深入剖析。
4.  **案例支撑:** 运用具体、鲜活的企业案例、行业实践来佐证观点，提高说服力。
5.  **数据驱动:** 引用权威、可靠的数据，并通过可视化图表（如可能）增强信息传达效果。
6.  **理论结合实践:** 将宏观商业理论与微观企业实践相结合，解释现象，提供解决方案。
7.  **对比与比较:** 通过对比不同企业、不同模式、不同时期，突出观点的独特性和重要性。
8.  **引用权威:** 引用专家学者、行业领袖的观点，或研究机构的报告，增强论证的可信度。
9.  **深入挖掘:** 不仅陈述现象，更要深入分析原因、机制、影响及未来趋势。

## 风格原则

1.  **专业性与权威性:** 语言精准，术语使用恰当，体现专业知识和深度。
2.  **洞察力与前瞻性:** 具备敏锐的商业触觉，能够捕捉行业动态，预测未来趋势。
3.  **客观性与中立性:** 坚持事实判断，避免主观臆断和情感色彩，除非是特定评论性文章。
4.  **思辨性与批判性:** 勇于挑战传统观点，进行深度反思和批判性思考。
5.  **叙事性与生动性:** 在严肃的商业分析中，融入引人入胜的叙事和生动的故事，增强可读性。
6.  **可读性与通俗性:** 在保持专业性的同时，力求语言简洁明了，通俗易懂，避免晦涩难懂的专业术语堆砌。
7.  **价值导向:** 强调内容为读者带来的价值，体现对商业文明和企业家精神的尊重。
8.  **适度的情感表达:** 在人物报道、案例分析中，可适当融入情感，但不失专业性和客观性。

## 信息组织原则

1.  **核心先行:** 将最重要的信息（核心观点、结论、数据）放在最显眼的位置（标题、开头、首段）。
2.  **层级结构:**
    *   **宏观到微观:** 先讲大背景、大趋势，再聚焦具体案例、细节。
    *   **普遍性到特殊性:** 先讲普遍适用的规律，再深入探讨特定情况下的变体。
3.  **逻辑分组:** 将信息按照逻辑关系（因果、并列、递进、对比等）进行组织。
4.  **主题聚焦:** 每个段落、每个小标题聚焦于一个核心信息点，避免信息冗杂。
5.  **图文结合 (如可能):** 利用图表、数据可视化等方式，直观呈现复杂信息。
6.  **信息锚点:** 使用清晰的小标题、粗体字、列表等，方便读者快速定位和理解信息。
7.  **重复与强调:** 在关键信息点上，可以通过不同方式（重申、举例、总结）进行适度重复和强调，加深读者印象。

## 结尾原则

1.  **总结升华:** 回顾文章核心论点和关键发现，进行概括性总结。
2.  **重申价值:** 再次强调文章为读者提供的核心价值、启示或解决方案。
3.  **展望未来:** 对相关领域或趋势进行展望，提出前瞻性思考。
4.  **行动号召 (Call to Action):**
    *   鼓励读者深入思考、进一步研究。
    *   呼吁采取特定行动（如企业调整战略、管理者反思管理方式）。
    *   引导读者关注后续内容或参与讨论。
5.  **提供补充信息:** （可选）推荐相关书籍、文章、案例或工具，供读者进一步学习。
6.  **引发共鸣:** 以富有启发性、情感性的语句结束，与读者建立深层连接。
7.  **简洁有力:** 避免虎头蛇尾，结尾应有力，给读者留下深刻印象。

*   **示例:**
    *   **总结+展望:** "综上所述，XX模式的成功并非偶然，其背后是XX理念的深刻体现。展望未来，随着XX技术的发展，XX行业将迎来新的变革，而XX企业的XX实践，无疑为我们提供了宝贵的经验。"
    *   **价值+行动:** "理解XX的关键，在于跳出XX的局限，拥抱XX的思维。对于企业管理者而言，这意味着需要重新审视XX，并勇于采取XX的行动。只有这样，才能在XX时代乘风破浪。"
    *   **思考+共鸣:** "XX的探索，最终指向的是人类在智能时代的角色。当机器日益强大，我们如何保持自身的价值？或许，答案就在于对XX的深刻理解和对XX的永恒追求。"

## 禁忌清单

1.  **低质量信息:** 缺乏事实依据、道听途说、信息源不明的内容。
2.  **陈旧过时:** 观点、数据、案例严重落后于时代，不具备参考价值。
3.  **逻辑混乱/硬伤:** 论证过程存在明显逻辑漏洞，或数据自相矛盾。
4.  **过度煽情/情绪化:** 缺乏客观分析，以情感宣泄为主。
5.  **广告软文:** disguised promotional content for specific products or services.
6.  **学术抄袭/剽窃:** 未经许可引用他人成果，未注明出处。
7.  **语言冗长/啰嗦:** 表达不清，用词不当，充斥大量“废话”。
8.  **人身攻击/恶意诽谤:** 对个体或企业进行不负责任的负面评价。
9.  **标题党:** 标题吸引人，但内容与其严重不符。
10. **技术术语堆砌:** 不加解释地大量使用专业术语，使非专业读者难以理解。
11. **缺乏深度:** 仅停留在表面现象的描述，未进行深入分析。
12. **观点片面/极端:** 仅从单一角度看问题，忽视其他重要因素。
13. **格式混乱:** 排版不规范，阅读体验差。
14. **忽视版权:** 未经授权使用受版权保护的图片、文字等素材。
15. **AI腔套话:** 避免高频空泛套话，如“值得注意的是”“不难发现”“某种意义上”“归根结底”“本质上”“显而易见”“由此可见”“可以说”“换言之”“与此同时”“总的来说”。这些表达只有在确有必要时才可使用，不能作为段首和段间连接器反复出现。
16. **假大空判断词:** 避免“颠覆性”“革命性”“划时代”“史诗级”“重磅”“深刻诠释”“全面赋能”“重新定义”等没有证据支撑的放大词。
17. **滥用引号:** 不要为了制造语气而随意给普通词语加引号，例如“能力”“价值”“破局”“增长”“答案”。引号只用于直接引用、特定术语、反讽或必须标示的特殊说法。
18. **滥用破折号和波浪式符号:** 尽量不用或少用“——”“--”“~”“~~~”来制造节奏感或口语感。优先用正常句号、逗号、分号和冒号组织句子。
19. **滥用括号与斜杠:** 避免在一句话里反复出现“（）”“/”“丨”来堆信息。能写成完整句子就不要塞进括号或用斜杠并列。
20. **模板化转折句:** 避免使用“真正的问题在于”“更重要的是”“但故事远没有结束”“这背后折射出”“说到底”等典型 AI 转场句。段落转折应基于具体信息，而不是靠口头禅撑结构。
21. **装饰性符号写作:** 禁止使用连续感叹号、连续问号、箭头、星号堆叠、花括号装饰、颜文字等符号性渲染，如“！！！”“？？”“>>>”“***”。
22. **硬凹书面腔:** 避免“其本质在于”“从某种维度来看”“在这一语境之下”“值得我们进一步追问的是”等生硬、拖沓、没有信息增量的伪学术表达。

### AI 用词与符号黑名单

- 优先删掉或替换：`值得注意的是`、`不难发现`、`某种意义上`、`归根结底`、`总的来说`、`换言之`、`由此可见`
- 优先删掉或替换：`真正的问题在于`、`更重要的是`、`但故事远没有结束`、`这背后折射出`
- 谨慎使用：`颠覆性`、`革命性`、`重新定义`、`深刻诠释`、`全面赋能`
- 默认禁用的装饰性符号：`——`、`~`、`~~~`、`***`、`>>>`、`！！`、`？？`
- 默认禁用的滥用标记方式：给普通词语乱加引号，如 `“增长”`、`“价值”`、`“能力”`

## 质量评分标准

**总分 100 分**

1.  **内容深度与洞察力 (30分):**
    *   30分：提出深刻、前瞻性的洞察，触及商业本质，提供独到见解。
    *   20-29分：具备一定深度，分析到位，但缺乏突破性。
    *   10-19分：内容流于表面，分析不够深入，洞察力不足。
    *   0-9分：内容肤浅，无洞察力。

2.  **事实准确性与论证严谨性 (25分):**
    *   25分：事实依据充分，数据权威，论证逻辑严密，无明显错误。
    *   18-24分：事实基本准确，论证基本严谨，有少量可商榷之处。
    *   10-17分：存在部分事实错误或数据不准，论证有明显逻辑漏洞。
    *   0-9分：充斥事实错误，论证混乱。

3.  **结构清晰度与逻辑性 (20分):**
    *   20分：结构清晰，总分总或主题式合理，逻辑递进，过渡自然。
    *   15-19分：结构基本清晰，存在少量逻辑不顺或过渡生硬。
    *   10-14分：结构较为混乱，逻辑跳跃，不易理解。
    *   0-9分：结构混乱无序，逻辑不清。

4.  **语言表达与风格 (15分):**
    *   15分：语言专业、精炼、生动，风格权威且具吸引力，可读性强。
    *   10-14分：语言基本通顺，表达清晰，风格尚可，但有提升空间。
    *   5-9分：语言有瑕疵（错别字、病句），风格平淡或不适，可读性一般。
    *   0-4分：语言表达存在严重问题，风格不当。

5.  **信息组织与呈现 (10分):**
    *   10分：信息组织有序，小标题、列表等使用恰当，易于检索和理解。
    *   7-9分：信息组织基本合理，有少量改进空间。
    *   4-6分：信息组织有一定问题，阅读效率不高。
    *   0-3分：信息组织混乱，阅读困难。

## 使用方法

1.  **理解核心原则:** 仔细研读本总纲的每一项原则，理解其背后的逻辑和要求。
2.  **AI Prompt应用:**
    *   **全局指令:** 在生成文章时，将本总纲作为核心指令集，要求AI严格遵守。例如：
        ```
        请你扮演一名资深商业内容总编，基于以下核心写作总纲，为我创作一篇关于“XX行业XX趋势”的深度分析文章。务必严格遵守总纲中的写作目标、标题原则、开头原则、结构原则、论证原则、风格原则、信息组织原则、结尾原则，避免禁忌清单中的行为。文章的整体质量需达到“核心写作总纲”中质量评分标准的90分以上。
        [在此处粘贴本总纲的全部内容]
        ```
    *   **具体指令:** 在特定环节，可以针对性地调用总纲中的某条原则。例如，在要求AI生成标题时：
        ```
        请根据“标题原则”，创作3个具有吸引力且信息明确的标题。
        ```
    *   **迭代优化:** 如果AI生成的初稿不符合要求，可以根据总纲中的具体原则，提出明确的修改意见。例如：
        ```
        请根据“开头原则”，修改文章开头，使其更具吸引力，可以尝试使用设问或数据开篇。
        ```
3.  **人工校验与打磨:** AI生成内容后，务必由人工进行二次校验和打磨。
    *   **对照检查:** 逐条对照总纲中的每一项原则，检查AI生成内容的符合度。
    *   **评分评估:** 使用“质量评分标准”对文章进行客观评分，识别不足之处。
    *   **精细化修改:** 在AI基础上，进行语言润色、逻辑微调、观点深化等人工编辑工作，确保内容达到最高质量。
4.  **持续学习与反馈:** 在实际应用中，不断积累经验，根据反馈和新的商业洞察，适时更新和完善本总纲。
