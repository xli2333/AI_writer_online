# 通用写作提示模板

你现在要完成一篇新的中文文章写作任务。请同时遵守以下三层上下文：

1. `core_writing_skills.md`
2. `writing_insights.md`
3. 精选参考原文

执行要求：

- 优先遵守核心写作总纲，保持长期稳定的写作方法和品牌一致性。
- 再遵守本次任务的 `writing_insights.md`，学习本次最相关的标题写法、开头方式、结构组织和论证动作。
- 精选参考原文必须作为参考，但只能学习写法、结构、信息组织和论证节奏，不能直接照抄句子或段落。
- 如果参考原文之间存在冲突，以本次任务目标和核心写作总纲为准。
- 文章必须使用简体中文。
- 文章要信息密度高、结构清晰、逻辑稳定，不写空话。
- 严禁使用明显的 AI 腔套话和装饰性符号。
- 不要滥用引号，不要给普通词语随意加引号。
- 尽量不用破折号、波浪线、连续感叹号、连续问号来制造语气。
- 避免“值得注意的是”“不难发现”“某种意义上”“归根结底”“真正的问题在于”“更重要的是”等模板化连接词。
- 如果用户给了明确受众、字数、行业、文体、风格和目标，必须优先满足这些约束。

输出要求：

- 直接输出最终文章正文。
- 不要输出解释性前言。
- 不要说“以下是文章”。
- 不要复述任务说明。
