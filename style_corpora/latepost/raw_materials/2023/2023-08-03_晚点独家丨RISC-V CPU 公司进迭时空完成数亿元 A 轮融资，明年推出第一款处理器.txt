《晚点 LatePost》 独家获悉，RISC-V CPU（中央处理器）创业公司进迭时空近日完成数亿元人民币 A 轮融资，融资将主要用于研发服务器 CPU 和引进人才。本轮融资由联想创投与君联资本领投，北京国管顺禧基金和老股东跟投。其中联想创投是联想集团的战略投资机构，联想目前是全球最大的个人电脑生产商和第三大服务器生产商。个人电脑和服务器都对 CPU 有大量需求。
进迭时空成立于 2021 年，此前已完成 3 轮融资，投资方包括经纬资本、耀途资本、万物资本、真格资本、海阔天空创投和趣合资本等机构，也包括香港科技大学教授高秉强、全志科技前总经理唐立华等学界与行业资深人士。
进迭时空的两名联合创始人陈志坚和孙彦邦，都在 RISC-V 领域工作多年，有开发大规模量产 RISC-V 芯片的经验。
陈志坚在 2006 年读博期间加入中天微，2018 年随中天微被阿里巴巴全资收购进入阿里旗下芯片公司平头哥，担任资深技术专家。他是平头哥 2019 年发布的国产嵌入式 CPU 玄铁系列的主要研发和应用负责人。阿里称，玄铁系列 CPU 累计出货量已超 25 亿片，主要应用于智能音箱、行车记录仪、打印机和安防摄像头等设备。
孙彦邦 2008 年加入全志科技，曾任全志科技 RISC-V 负责人、全志在线有限公司总经理。全志的芯片大量应用于机顶盒、扫地机器人和平板电脑等产品中。在职期间，孙彦邦带队在 2021 年量产了全志第一颗 RISC-V CPU D1 ，D1 主要应用于显示器、摄像头和智能音箱等产品上。
目前进迭时空有 130 多人，主要成员来自平头哥和全志科技等公司。进迭时空计划于明年推出第一款产品 RISC-V 处理器核 “X100”，其目标场景是高端智能机器人、边缘计算卡和边缘服务器等。
主要指令集中，唯一开源的一个
进迭时空成立的 2021 年底是上一轮芯片投资热潮的尾声。这轮热潮始于 2016 年的 AI 芯片，最初的动力是 AI 软件技术变革带来了新的硬件需求；2018 年之后，中美科技竞争又进一步刺激国内大芯片创业热，GPU（图像处理器）、DPU（数据处理器）和 ARM 架构 CPU 公司一浪连一浪出现。
而到 2021 年-2022 年，国资和人民币基金在政策导向下，将重心转移至半导体上游的制造与装备投资，加上更严酷的外部环境，芯片设计融资大幅减少。据企名片数据，去年下半年至今年上半年，芯片设计一级市场共完成约 177 起投融资交易，同比下滑 40%。
陈志坚说，在这个时间点创业有三个原因：一是自己更成熟了，二是找到了合适的联合创始人与团队；最重要的是，RISC-V 是中国半导体行业的大机会，应该有人站出来抓住它。
RISC-V 最早于 2010 年前后由美国加州大学伯克利分校的大卫·帕特森（David Patterson）教授团队设计。它是一种芯片指令集，规定了芯片和软件的设计规范。
目前世界上主流的指令集是 x86 和 ARM。RISC-V 和 ARM 同属精简指令集，x86 是复杂指令集。精简指令集的优势是可以把芯片做得更小，降低制造成本，且功耗较低。但它处理复杂计算的能力相对弱。基于 x86 的 CPU 现在主要用在个人电脑和数据中心里的服务器上，基于 ARM 的 CPU 主要用在移动端设备，如手机上。RISC-V 芯片目前主要服务物联网设备，如智能门锁、音箱和摄像头等。
RISC-V 最大的特点是免费与开源，可以被自由地用于商业目的。任何人都能基于 RISC-V 设计、制造和销售芯片，不受单一国家或公司制约，也无需支付 IP 授权费。而 x86 由英特尔和 AMD 等几家公司控制，不对外开放。ARM 则由英国公司 Arm 掌握，芯片设计公司要基于 ARM 设计芯片，需向 Arm 支付高昂授权费。
2015 年 RISC-V 基金会成立，这个新的开源指令集从高校进入产业界，随后陆续有大公司和创业公司开始基于 RISC-V 开发芯片。
RISC-V 主要发明人之一克斯特·阿萨诺维奇（Krste Asanovic） 在 2015 年成立了芯片设计公司 SiFive。参与苹果 A 系列、AMD Zen 架构设计的知名芯片工程师吉姆·凯勒（Jim Keller）也在 2021 年投身 RISC-V 通用 CPU 创业，加入了加拿大 AI 芯片创业公司 Tenstorrent。欧洲、日本、中国台湾等多个国家和地区也有相关公司出现。
凯勒多次表达他对 RISC-V 开源生态的信心，他认为 RISC-V 是更年轻的架构，没有太多前人添加的 “垃圾”，干净、高效。“RSIC-V 未来会获胜。” 今年 6 月接受 EE Times 采访时他说。
开源除了因本身的开放性受到凯勒等技术人士的支持外，它对谋求更多自主性的中国半导体产业还有特别的意义。
自 2018 年起，中国的 RISC-V 生态开始加速发展。中科院计算所牵头成立中国开放指令生态（RISC-V）联盟，成立 AI 芯片公司中科物栖；达摩院也在这一年将芯片研发团队与中天微团队合并成立平头哥，开始研发 RISC-V 芯片玄铁。

据 RISC-V 基金会数据，截至 2022 年底，RISC-V 芯片全球出货量已达 100 亿颗，其中约 50% 由中国公司设计。
未来空间有多大，取决于能否进入更高端的市场
RISC-V 当下的技术趋势是从性能低、价格低、架构简单的嵌入式芯片，向性能要求和技术壁垒都更高的大算力芯片发展。
这与同属精简指令集的 ARM 架构走过的路类似。高通等公司开发的 ARM CPU 最初在手机市场战胜了 x86，成为主流。2018 年之后，更多公司尝试用 ARM 开发个人电脑、服务器 CPU 等更高性能的产品，这本是 x86 的优势地盘。进攻成果已经显现：苹果电脑自 2020 年开始使用的 M1 芯片就是 ARM 架构，亚马逊也在 2018 年推出了基于 ARM 的自研服务器 CPU Graviton。这证明了精简指令集能做出高性能芯片。
ARM 的先例之后，更多公司开始基于开源、免费的 RISC-V 寻找设计高性能芯片的机会。陆续有 Google、算能科技和 Ventana Micro Systems 等公司尝试研发 RISC-V 服务器 CPU，但其实际市场份额还很低。据市场调研机构 Counterpoint 的数据，2022 年全球服务器 CPU 市场中，约有 91% 的芯片为 x86 架构，ARM 和其他架构的占比分别为 8% 和 1%。
陈志坚认为，RISC-V 与 ARM 都会走向更高性能的市场，但路径稍有区别：ARM 在手机之后的下一站是个人电脑，再之后是服务器；RISC-V 在智能物联网设备后，第二步会先进入数据中心的服务器市场，然后是手机和个人电脑。他判断，RISC-V 进入服务器领域需要 5 年左右时间，而进入个人电脑和手机可能需要 10 年时间。ARM 在手机上普及后先进入个人电脑，与苹果和 Arm 的多年紧密合作有关。而 RISC-V 最早从物联网做起，先做服务器，积累生态、技术和经验后，再进入生态壁垒很高的手机市场是更可行的路径。
物联网设备之外的更多场景，将大幅扩大 RISC-V 的市场空间，因为即使卖出海量的低端芯片，其收入、利润和以利润支撑的研发投入，仍无法与能设计主流高端芯片的顶尖公司相比。
RISC-V 到底能不能，或者会以何种方式、节奏进入更高端的芯片市场，尚没有共识。一种行业观点是，RISC-V 现阶段还是更适合做物联网芯片等相对低端的产品。“要做好大算力芯片，核心不是架构开源与否，而是生态。另起炉灶的架构短期内难以形成竞争力。” 一位芯片投资人说。同时有声音认为，RISC-V 创业潮的出现，尤其是用 RISC-V 做高性能芯片的实践，主要由政策推动，而不是因为市场与技术已经准备好了。
不论时间和路径如何，大部分 RISC-V 从业者都相信这种开源指令集最终会进入更高端的芯片市场，与封闭指令集分庭抗礼。
今年 3 月，RISC-V 之父大卫·帕特森在玄铁 RISC-V 生态大会上说，RISC-V 最有机会发力的是人工智能和机器学习领域，在这些场景中，它的复合增长率将超过 70%。
吉姆·凯勒在今年 8 月的一个最新采访里说：开源很酷的一点是，一旦人们开始为它贡献，它就会成长，这是一条单行道，“当你习惯了开放，就再也回不去了”。
