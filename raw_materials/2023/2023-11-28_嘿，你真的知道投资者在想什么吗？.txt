了解投资者的需求，要从投资者和公司的沟通和对话开始。虽然上市公司会通过各种报告披露相关信息，但投资者在尝试理解和处理这些信息并做出投资决策时会面临一系列问题。投资者互动平台提供了一个沟通媒介，让上市公司可以了解和回答投资者真正关心的问题，从而影响他们的投资决策，提高市场流动性以及股价中的信息量。
与投资者沟通是每个公司的必修课，尤其是上市公司，要对所有股东负责。上市公司每年都要公开自己的财务报表、战略规划等，给投资者接下来的投资决策提供一定参考依据。随着资本市场监管的完善，上市公司披露的信息更丰富。然而，上市公司每年披露的公报是否足以向投资者说明公司的运营状况？投资者真正想知道的是什么？他们在处理信息的过程中会遇到什么样的问题？公司的新闻如何影响投资者的决策？这些信息又如何影响公司的股价？
对于上市公司而言，这些问题是亟待了解的黑匣子。尽管上市公司已经披露了更多信息，但真正被投资者纳入投资决策的信息却十分有限。过去学术界关于公司披露问题的研究更多关注供给方，也就是公司，却很少从需求方，也就是投资者的角度看待披露问题。
了解投资者的信息需求以及信息处理困境，对于资本市场的健康运行至关重要。由投资者发起的双向沟通机制，则是探究这一问题的重要切入点。
清晰的沟通往往是通过各方之间的持续对话来实现的。了解投资者的需求，同样要从投资者和公司的沟通和对话开始。投资者互动平台（interactive investor platforms）为了解上述问题带来了全新的视角。
从2010年起，中国两大证券交易所——深圳证券交易所和上海证券交易所——各自推出了自己的投资者互动平台：“互动易”和“e互动”，目的是在投资者和上市公司之间建立直接沟通的桥梁。在投资者互动平台上，每家上市公司都有自己的社区，投资者可以直接在目标公司的社区页面上发布他们的问题，由公司管理层回答。参与的公司必须指定一名管理层人员来监督公司的答复。虽然这些平台的运营受到证券交易所的监管，但每家上市公司对自己的回复负有法律责任。一旦用户提交了一个问题，整个沟通过程就受到证券交易所的控制，被提问的公司没有权利编辑或删除问题。问题一经发布，公司和公众能同时看到。平台上的问题都由投资者提出，更能凸显投资者真正关注的信息。
投资者想知道什么？
2010-2017年，中国投资者在投资者互动平台上发布了约250万个问题，绝大多数能在几天或几周内得到公司管理层的正式答复。截至2017年年底，超过99%的中国上市公司都在平台上与投资者互动。
研究者使用网页内容抓取技术，从这些平台收集了2010年1月-2017年12月的全部互动内容，然后通过BERT自然语言处理模型对帖子的性质和内容进行分类。结果显示，绝大多数帖子的内容是寻求对特定项目的答案或解释（79. 8%），其次是向管理层提出建议或其他意见（16.6%），以及要求核实/否认谣言或纠正误解（2.85%）。从问题的具体内容来看，投资者询问最多的10个话题依次为产品或企业经营（21. 7%）、财务报告（18. 6%）、公司治理（9. 0%）、股票交易（7. 1%）、资产重组（6. 6%）、投资（5. 9%）、融资（4. 8%）、分红（4. 4%）、行业相关问题（4. 4%）、内幕交易（4. 1%）。研究者列举了10个投资者和公司的互动案例来对这一问题进行深度解析。
案例1：具体项目
( 信息已公开，但投资者不确定会计处理方法)
科陆电子 [ 股票代码：002121]
问题：
公司近期因被收购企业未达到业绩承诺而获得补偿(约780万元)，这笔补偿能否在净利润中体现?
答复：
是的，这780万元会在2023年的净利润中体现。
案例2：具体项目
( 报告的风能和光伏订单总数)
九洲电气 [ 股票代码：300040]
问题：
不好意思，我老了，看不清你那一大堆公告了。我想问一下，2017年你们签署的与风电和光伏发电相关的订单人民币总额是多少? 考虑到对风电和光伏发电的监管收紧，请对相关风险进行管控。
答复：
谢谢您的关注！2017年，我公司已签订了约25亿元人民币的风电和光伏发电合同。
案例3：行业相关
（投入品价格上涨对业绩的影响）
格林美 [ 股票代码：002340]
问题：
最近钴和钨价格的上涨导致硬质合金生产商的投入成本上升。请谈谈这对公司硬质合金业务的潜在影响。
答复
：
谢谢您的关注！硬质合金行业是一个相对成熟的行业，年增长率为8%-10%。今年初以来，钴、钨价格上涨，特别是钴价格上涨，推高了硬质合金产品的投入价格。但是，硬质合金产品中钴的平均含量只有15%左右，所以它的价格变化对我们最终产品的价格影响不大。此外，我公司回收硬质合金废料作为钴和钨原料的一部分，确保了硬质合金业务中主要金属的有效回收。
案例4：谣言相关
（要求澄清传言中的债务隐瞒问题）
金城医药 [ 股票代码：300233]
问题：
有媒体指责贵公司在收购北京朗依制药有限公司( 以下简称“朗依”) 的过程中隐瞒了巨额债务。朗依因拖欠7100万元建筑费用而被起诉。贵公司为何不披露这一重大信息?
(报道链接：http：//finance.sina.com.cn/roll/2017-04-28/docifyetwtf8843011.shtm)
答复：
亲爱的投资者，感谢您的关注。朗依及其原股东的相关信息已经在我们的重组报告中披露。关于报道中提到的诉讼，双方已经达成和解，案件已撤诉。我司也在诉讼进度报告中提供了详细的和解方案。如果此事件给公司造成任何不可预见的损失，我们将向朗依原股东追索，防止投资者的利益受到侵害。作为一家上市公司，我们始终对媒体监督持开放态度。但是，如果出现不负责任或恶意的报道，我们将采取法律行动来保护我们的声誉和投资者的利益。
案例5：误解
( 对并购溢价计算的困惑)
中国长城 [ 股票代码：000066 ]
问题：
根据评估报告，中国电子财务有限公司( 以下简称“CEC”) 的账面净资产为27.72亿元，因此公司15%的股份价值4.158亿元。计划中的收购价格约为5.07亿元，溢价超过20%。请解释为什么会有这么高的溢价，以及以这么高的价格购买CEC 15%股份的动机。
答复：
各位投资者好！根据立信会计师事务所出具的审计报告， CEC的股权账面价值为28.43亿元，根据对可比案例的评估，确定其担保市净率为1.19。因此，其股权价值为33.83亿元。在此基础上，确定CEC 15%股权转让价格为5.074 6亿元。详情请参考CNINF网站上的评估报告。此次收购的目的是充分利用CEC的金融平台，实现有效的资金管理，通过优化资产组合和业务结构，加速公司的成长。
案例6：误解
( 混淆债务担保与未偿债务)
亿纬锂能[ 股票代码：300014]
问题：
公司为子公司21.6亿元的债务提供担保。然而，该公司的总债务只有22.9亿元，这似乎是不合常理的。你能解释一下吗?
答复：
公司签订了财务担保协议，根据该协议，如果子公司违约，公司有义务偿还最高21.6亿元的债务，包括银行授信协议、与荆门高新技术产业投资有限公司的贷款合同、设备进口信用证。在授信额度内，子公司可根据自身需要申请融资。因此，担保不属于公司的贷款或债务，不会反映在债务账目中。有关公司借款的详细情况，请参考公司年度财务报表。
案例7：误解
( 对税率和申报税额的混淆)
华铁股份 [ 股票代码：000976]
问题：
从年度报告中，我发现利润总额为9200万元，企业所得税为6355万元，这意味着税率接近70%！为什么?
答复：
利润总额为与通达集团( 香港) 有限公司( 以下简称通达) 合并后的数据。母公司报告利润为负，因此不必缴纳企业所得税。财务报告中的6355万元是指通达的子公司从合并日起至当年年底期间缴纳的所得税，通达子公司的总利润为3.468 7亿元，说明税率低于20%。
案例8：不合理的要求
( 未披露信息)
国光股份 [ 股票代码：002749]
问题：
我想知道公司今年是否会支付股息。只有支付现金股息才能使股价保持在较高水平，但可能导致投资者对公司失去信心。
答复：
根据深交所关于互动平台的相关规定，我们本着认真负责的态度积极与投资者沟通，但在沟通过程中不能泄露任何未披露的重大信息。所有强制性信息将按照信息披露要求，准确、完整、及时地在指定平台( 如http ：//www.cninfo.com.cn/) 上披露。
案例9：不合理的请求
( 未披露信息)
大恒科技 [ 股票代码：600288]
问题：
请详细披露科技部支持的两个项目的进展情况和计划。
答复：
我公司已经在财务报告中披露了相关信息。对于未披露的信息，根据上海证券交易所关于e互动的规定，我们不能在互动平台上与投资者沟通。
案例10：建议
京运通 [ 股票代码：601908]
问题：
在我看来，贵公司很有发展前途。同时，我想给贵公司一些建议。1 ) 除了电站之外，研究太阳能在太阳能汽车、电动汽车甚至飞机等领域的应用。2 ) 开展石墨烯在太阳能领域的应用研究。具体来说，石墨烯与晶体硅的结合可以达到更高的效率，从而实现未来对晶体硅的全面替代。3 ) 加快发展环境事业，特别是在京津冀地区。
答复：
嗨，亲爱的投资者，您的反馈和建议将被提交给公司的相关领导。谢谢您！
有许多提问是关于特定交易的澄清：投资者听说了某些影响公司的事件，并寻求澄清其影响。这些投资者通常已经了解到一些公开信息，但无法根据已有信息获得结论。背后的原因有很多，例如投资者不确定事件给公司带来的实质性影响（案例1），或者投资者不想去做更多的功课，想从公司这里快捷地获取答案（案例2）。有些时候，投资者关注到外部环境变化，想知道这种变化如何影响公司业务（案例3）。
对这类问题，管理层可以用涉及发布新信息作为理由来回避，但该公司还是引用了一系列公开事实来回答这个问题。研究者认为，该公司希望利用这个机会缓解投资者对潜在破坏性消息的担忧。同样地，在案例4中，公司通过平台上的回答来平息关于隐瞒债务的谣言。
第二类频繁出现的问题涉及投资者对某些事件或交易的会计处理方法的疑惑，被归类为投资者的“误解”。在案例5中，投资者通过一份评估报告得出结论，该公司并购溢价过高。公司因此解释了多重估值的基本原理，证明其支付的溢价的合理性。在案例6和案例7中，投资者提出了对公司负债总额和纳税额的质疑，公司解释了数据的计算方式。
按照规定，公司不得在投资者互动平台上透露或者发布尚未公开的重大事件信息，因此，公司会拒绝回答一些不恰当的问题（案例8和案例9）。在案例10中，投资者提出的不是问题，而是对公司运营的建议，这一类型的帖子不需要公司进行回答或解释。
与公司互动对投资决策有影响吗？
从上述案例可以看出，投资者在尝试理解和处理公共信息并做出投资决策时会面临一系列问题，其中大部分投资者都希望在理解公司运营和财务数据方面获得帮助。
虽然投资者互动平台不能用于发布新信息，但在平台上发生的管理者和投资者之间的对话能够帮助投资者处理已有信息，从而缓解投资者的担忧，刺激交易。
为了验证投资者平台上的互动对投资者决策的影响，研究者以平台上的互动活跃度（包括发布的问题和回复的数量以及这些问题和回复的长度）为自变量，记录了异常交易量以及绝对异常收益的变化。在控制收益公告、盈利预测、重大事项报告、研究报告、媒体报道等其他信息披露事件后，结果显示，两者之间存在强烈的正相关关系，也就意味着，
投资者互动平台上的活跃度越高，引发的市场反应越强，会带来越高的交易量和回报波动率。
与投资者互动对公司利益有影响吗？
上市公司加入互动平台带有“半强制”的意味。尽管证券交易所没有明确要求所有上市公司必须回答投资者提出的所有问题，但设定了多个“投资者友好”指标，对表现出色的公司给予嘉奖。这就将投资者互动平台与社交媒体平台、聊天室、博客等其他非正式信息披露渠道区分开来。
投资者互动平台的另一个重要功能就是能够帮助投资者实时跟进公司动向。投资者只要在平台上关注了一家公司，一旦该公司的社区中发布了新的问题或回答，投资者都会收到提示。这些信息能够帮助投资者更深入地了解公司的运营状况，减少信息的不对称问题。研究发现，在投资者互动活跃度较高的时候，平均买卖差价较低，
这意味着充分的信息沟通降低了信息不对称带来的风险，改善了市场流动性。
当市场流动性得到改善，信息更容易体现在价格上，从而增加价格中有关未来收益的信息量。研究者分析证实，投资者和管理者在互动平台上发布信息的频率越高，累积预盈公告回报与未来标准化预期外收益的正相关关系越显著。也就是说，投资者互动平台活动增加有助于提高股价中的信息含量。
无论是提高市场流动性还是股价中的信息量，其底层逻辑都是要降低投资者信息处理成本。在日常生活中，投资者关心的问题大多可以通过整合已有的公开信息解答，投资者互动平台提供了一个满足投资者信息需求的方式。

投资者互动平台：
消除信息迷雾，增进投资者信心
对于投资者来说，企业的经营状况和他们的个人利益挂钩，因此他们非常关注公司的财务报告、
公司治理、股票交易、资产重组、投融资、分红、行业动态以及内幕交易等问题。有时候上市公司可能会觉得困惑：这些信息明明已经在年报中披露，为什么投资者还是不了解？
事实上，并不是所有投资者都有足够的信息意识和整合能力，甚至有些投资者就单纯地不愿意花时间去阅读、分析这些信息，想要直接获得答案。
不论是何种情况，投资者互动平台都是一个能帮助上市公司澄清误解、缓解担忧、提振信心的渠道。
因此，
管理者需要重视对投资者互动平台的管理，当有投资者提问时，管理者需要迅速、详细
地解答和回应这些问题。
尤其是当媒体发布了可能与公司相关的新闻或者信息之后，管理者要有预期：一定会有投资者在平台上寻求公司的解答。公司及时给出准确的回答有助于减少信息不对称的风险，降低外部因素对公司股价的影响。
